## Q1. 本时段讨论最多的交易标的

统计窗口内被提及的交易标的(BTC / ETH / SOL / 各种 $TICKER / 项目名等),输出:

- **热度 Top(至少前 5)**:每个标的 + 提及次数 + 主要讨论方向(利好 / 利空 / 纯分析 / 喊单 / 避险)
- 每个标的至少给 **2 条代表性推文链接**(带 `@handle`)以便回溯
- 窗口内若无集中讨论,如实写"无明显热点标的",别硬凑

**"提及"的判定**:标题行 author、正文、`🔗 quote:` 里的完整正文、`🖼️ article: "..."` 里的标题——都算。

**计次规则**:每条 block(独立 statusId)算 1 次;`🔁 reposted by: @A, @B` 一条 block 里多个 reposter **加总**到该标的的提及次数(因为这是信号强度);quote 不独立计次(已属于外层 block)。

## Q2. 有价值的交易策略分享

窗口内有没有**具体、可操作**的交易策略分享?判定门槛:

- 明确到 **标的 + 方向(多 / 空) + 入场位 / 止损位 / 目标位** 中至少 2 项的,算"高信号策略"
- 只给观点没给执行条件的,**降级**成"情绪判断",可提一句但不当核心素材
- 纯喊单("梭!""冲!"但没依据)**不算**
- 带 chart 截图(raw 里 `🖼️ N images`)且有文字分析的,算加分项

**输出每条策略**:
- 推主 `@handle` + 策略一句话 + 推文链接
- 若策略有前提假设(如"BTC 站稳 70k 才入")明确写出
- 窗口内若无,写"无明确策略分享"

## Q3. 新上市 / 即将上市二级的币

窗口内有没有讨论**即将上 CEX 二级市场 / 刚上二级**的币?(IEO / ILO / 新 listing / 空投解锁 / TGE 等)

对每个提到的币:
- **标的名** + 上市所 / 上市阶段(如 Binance 现货 / Upbit listing / Bybit Launchpool)
- **多方观点汇总**:

  - 看跌的理由(同上)
- **净情绪倾向**:整体偏多 / 偏空 / 分歧明显 / 信号不足
- 窗口内若无,写"无新 listing 相关讨论"
