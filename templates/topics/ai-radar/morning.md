# 🌅 早间简报生成 Prompt

## 任务

基于 inputs({DATE} 的 raw 采集),生成早间时段简报。

## 数据范围

输入文件:
{INPUTS_LIST}

**报告范围**:{DATE} 当天 raw 中**所有 block**。早间报告聚焦"快速跟上节奏",不做严格时间过滤 —— raw 里有什么就分析什么。下面所有问题里出现"窗口内""本时段"都指当前 inputs 的全部内容。

如果 raw 文件里 block 时间戳跨日(比如凌晨补抓昨晚的),也按归属日 {DATE} 处理(raw 文件名即是归属日)。

## 聚焦题目(早间版 7 题)

早间只关注跟得上节奏需要的 7 题,不做全量分析。所有"窗口内"都指上面定义的窗口。

### Q1. 最高声量事件

窗口内跨 3+ 数据源同时出现的事件是什么?
(数据源 = 多位 KOL 独立转发/评论、原推高 RT 数、被引用讨论)

输出:事件一句话 + 信号强度标注 + 核心 3 方观点 + Takeaway

### Q2. KOL List 的辐射中心

窗口内被 List 里最多人引用/转发的**单条推文**是什么?
这往往是当前讨论的"原点",值得顺着扒完整语境。

输出:原推内容 + 辐射范围(被谁转/评) + 为什么重要

### Q3. 重要产品 / 模型发布

窗口内有哪个产品或模型的 release 引发了技术性讨论(非"好厉害"式转发)?

输出:发布内容 + 评论区的专业争论点 + Takeaway(是否值得做技术解读)

### Q6. 翻车 / 被打脸事件

窗口内有谁被公开纠错 / 群嘲 / 道歉?
如无,此题可跳过。

### Q14. 中英文温差(海外视角)

窗口内英文圈热议但**中文圈还没跟上**的话题?
这是中文博主 48 小时抢首发的窗口。

输出:话题描述 + 英文声量 + 建议中文化切入角度 + 预估窗口剩余时间

### Q15. 早期微弱信号

窗口内讨论度还不高但"气味对了"的论文/项目/观点?
(AK 推荐的新 arxiv、小众但高质量的技术文、冷门但可能爆的想法)

输出:项目/观点 + 为什么有潜力 + Takeaway(是否值得做前瞻稿)

### Q17. 窗口期预警

综合以上,今天有没有"48 小时不写就过气"的选题?

## 输出格式

```markdown
# 🌅 AI Radar morning 简报 — {DATE}

**归属日**: {DATE} | **采集源**: {SOURCES} | **分析推文**: {N 条}

---

## Q1. 最高声量事件

Signal: 🟢🟢🟢 / 🟢🟢 / 🟢 / ⚪

{答题正文}

**Takeaway**: {一句话,能直接落到"我今天该不该聊这个"}

---

## Q2. KOL List 的辐射中心

Signal: ...

{...}

---

[依此类推到 Q17]

---

## 📬 早班 Takeaway 汇总

- ⚡ 不能错过:{1-2 条当前必须知道的事}
- 🎯 中文首发窗口:{Q14 里最值得抢的 1 个}
- 🔮 潜在前瞻选题:{Q15 里值得收藏的 1 个}
```

## 写作风格

- **紧凑**:每题 3-6 行,不啰嗦
- **数据优先**:有具体数字(RT/评论/star)就标出来
- **带链接**:推文、项目必带 source URL,便于你点开看原文
- **不重复**:如果 Q1 和 Q3 聊的是同一件事,后者指向前者即可

## 判定规则

- **Signal 强度**:🟢🟢🟢 = 跨 3+ 源独立验证 / 🟢🟢 = 2 源 / 🟢 = 1 源但有独立判断价值 / ⚪ = 数据不足
- **Takeaway 不能水**:写不出具体建议的题,老实写"无特殊行动建议"
- **严格基于 inputs**,不要引入你训练知识 / 外部信息编造背景

## 写入方式

生成完整 markdown 后,**用 Bash heredoc 管道给 wiki-write 脚本**,它会对当日 wiki 的 `## section: {SECTION_NAME}` 段做幂等 upsert(其他 section 原样保留,同 section 重跑替换自己那段):

```bash
{WIKI_WRITE_CMD} <<'PERCH_EOF'
(把你上面"输出格式"里生成的完整 markdown 原样放这里,**不要**再包 `## section: {SECTION_NAME}` 外层标题,脚本会自动加)
PERCH_EOF
```

最终文件落在 `{WIKI_PATH}`(当日所有 section 共用一份)。不要自己用 Write 工具直接覆盖,会破坏其他 section。
