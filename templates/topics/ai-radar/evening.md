# 🌆 全天主报告生成 Prompt

## 任务

基于 `{RAW_PATH}`（{DATE} 完整 24 小时数据），生成全天主报告。

## 时间窗口

**完整 24 小时**（今日 00:00 ~ 20:00，加上昨晚剩余时段）。

## 定位

**这是用户真正用来定选题的主产出**。早间和午间是"不错过"，晚间才是"决定做什么"。

需要严谨、完整、给出可执行建议。

---

## 两层维度

- **Layer 1 信息层（Q1-Q15）**：有什么有意思的信息（客观）
- **Layer 2 创作建议层（Q16-Q17）**：今天能写什么（主观 + 可执行）

---

## Layer 1 — 信息层（15 题）

### 🌡️ Pulse 话题脉搏

**Q1. 今日最高声量事件**
今天在 X List 中跨多源同时出现的事件是什么？单点爆发不算，真正"出圈"的才算。

**Q2. KOL List 的辐射中心**
今天被你 List 里最多人引用/转发的**单条推文**是什么？这往往是今天的讨论原点。

**Q3. 产品 / 模型的深度发布**
今天有哪个 release 引发了技术性讨论（不是"好厉害"式转发）？评论区在争什么？

### ⚔️ Conflict 观点战场

**Q4. A vs B 的清晰对撞**
今天两方在同一问题上的公开分歧？各方核心论据？

**Q5. 反共识观点信号**
今天哪位 KOL 发了明确的反主流观点？论据站得住吗？

**Q6. 翻车 / 被打脸事件**
今天有谁被公开纠错 / 群嘲 / 道歉？

### 📖 Story 案例故事

**Q7. "某人用 AI 做了 X" 的故事**
今天最有故事相的个人案例？要有细节 + 反转 + 数字。

**Q8. 带真实数据的公开分享**
今天有谁公开了具体数字——营收、用户数、token 消耗、转化率、A/B 结果？

**Q9. AI 辅助 / 取代工作的具体案例**
不是笼统"AI 会取代 X"，要具体到"某工种 + 具体流程 + 具体结果"。

### 🛠️ Craft 实用干货

**Q10. 被多人验证的 prompt / 工作流**
今天有没有一个技巧被 2+ 个 KOL 独立验证有效？（教程稿素材）

**Q11. 适合"实测对比"的新品**
今天新出的模型/工具里，哪个做"我用它 vs 我用 X"横评会有流量？

**Q12. 可整理成清单的小技巧集**
今天零散但值得攒起来的小 tip？（攒到周末出清单稿）

### 🏢 Enterprise 企业落地信号

**Q-E. 解决企业实际问题的产品 / prompt / skill / 思想**
今天有没有哪个内容是在解决**具体企业场景**的问题？
（营销获客 / 降本提效 / 决策辅助 / 研发加速 / 客户服务）

这是用户独特的差异化定位（AI 博主 + 商业咨询），单独列出。

### 🔍 Edge 信号

**Q13. 出乎意料的冷 / 热反差**
今天什么东西"应该火却没火"或"不该火却火了"？反差本身就是钩子。

**Q14. 中英文温差**
英文圈热议但中文圈还没跟上的话题？（48h 抢首发窗口）

**Q15. 早期微弱信号**
今天讨论度不高但"气味对了"的论文 / 项目 / 观点？

---

## Layer 2 — 创作建议层（2 题）

### 🎯 Q16. 今日一篇选题推荐（重点）

基于今天全部信号，结合用户的 3 种文章类型，推荐 1-3 个今日最优选题。

用户的 3 种文章类型：
1. **碎碎念**（猫笔刀式）：日常盘点 + 大白话 + 微观点
2. **热点专题攻略**：切入点是"落地结合实际场景"
3. **AI 深度思考**：AI × 现实世界结合

对每个推荐选题给出：

```
### 选题 1: {一句话核心}
- **文章类型**: {碎碎念 / 攻略 / 深度}
- **推荐理由**: {为什么是今天写这个，不是明天}
- **差异化切入点**: {避开大路货，你的独特角度}
- **标题候选**:
  1. {标题 A}
  2. {标题 B}
  3. {标题 C}
- **核心钩子**: {一句话抓人点}
- **推荐开篇**: {2 句开头备选}
- **核心素材**: {来自今天的哪 2-3 条推文 + 链接}
```

至少推 1 个，最多 3 个。不够就不硬凑。

### ⚠️ Q17. 窗口期预警

今天有没有"48 小时不写就过气"的选题？
（热度刚起 + 深度稿还没出现 + 有先发优势）

如有，独立列出，标红。

## 严格要求

1. **每题必有 Signal 强度标注**
2. **每题必有 Takeaway（Q16/Q17 除外，它们本身就是 Takeaway）**
3. **Q16 是重头戏**：差异化切入点必须具体到"和大路货的区别在哪"
4. **不够就不凑**：某题没信号就标 ⚪，不要编造
